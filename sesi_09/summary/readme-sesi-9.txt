--Single Page App with React Router--
>SPA = implementasi web yang memuat hanya satu dokumen web
>konten isi dokumen tunggal tsb diperbarui melalui API JS 

--SPA Life Cycle--
>SPA tidak memerlukan reload page ketika digunakan
>user tidak akan berpindah halaman dengan melakukan request kepada server setiap kali terjadi interaksi pada apliksi
>SPA hanya akan melakukan load terhadap satu halaman dari server, kemudian mekanisme routing di handle dan dibebankan kepada client

--React Router--
3 komponen utama dalam implementasi React Routers

1. Router <BrowserRouter>
2. Route <Route> dan Routes <Routes>
3. Navigation <Link> , <NavLink>, <Redirect>

--Programmatic Navigation--
>digunakan untuk navigasi secara dinamis
